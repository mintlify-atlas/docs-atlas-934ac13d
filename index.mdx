title: "Flipper Zero Unleashed Firmware"
description: "The most stable custom firmware for Flipper Zero — unlocked Sub-GHz frequencies, new protocols, enhanced NFC, JavaScript scripting, and more."
---

Unleashed Firmware is a community-driven fork of the official Flipper Zero firmware. It removes regional Sub-GHz restrictions, adds dozens of new protocols, expands NFC support, and includes quality-of-life improvements — all while staying fully compatible with the official API and app ecosystem.

<Warning>
  This software is intended solely for experimental and educational purposes. Use only within the bounds of applicable law. This project is developed independently and is not affiliated with Flipper Devices.
</Warning>
