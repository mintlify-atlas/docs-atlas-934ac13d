Unleashed Firmware extends the stock protocol list with dozens of additional static and rolling-code decoders. The table below covers the default Sub-GHz app only. Apps such as Weather Station maintain their own separate protocol list.

<Note>
  `433 MHz` in this list means `433.92 MHz`. `868 MHz` means `868.35 MHz` unless otherwise noted.
</Note>

## Modulation types

Understanding the modulation column helps you pick the correct preset when reading an unknown remote.

<AccordionGroup>
  <Accordion title="AM650 (OOK, amplitude shift keying)">
    The most common modulation for Sub-GHz remotes. The carrier is switched on and off to represent bits. AM650 refers to the 650 µs bandwidth filter used during reception. The vast majority of garage, gate, and alarm remotes use this modulation.
  </Accordion>
  <Accordion title="FM / FSK (frequency shift keying)">
    The carrier frequency shifts between two values to represent 0 and 1. In the Flipper Sub-GHz menu, FSK presets appear as `FM476`, `FM12K`, etc. — the number indicates the frequency deviation in Hz. When a protocol lists `FM`, try each FM preset in **Sub-GHz → Read → Config** until you get a decode. `FSK476` in the protocol list corresponds to `FM476` in the menu.
  </Accordion>
</AccordionGroup>

## KeeLoq rolling-code manufacturers

KeeLoq is a proprietary rolling-code encryption system used across the majority of European and some North American gate and garage systems. Unleashed Firmware supports full decode and emulation for the manufacturers below.

<Note>
  Unless otherwise noted, all KeeLoq entries operate on **433.92 MHz** with **AM650** modulation and carry a **64-bit** packet. The default serial learning type is `simple` with a 10-bit serial part in the Hop field.
</Note>

## Protocol type reference

| Type | Description | Examples |
|---|---|---|
| Static | Fixed code — identical transmission every press | Princeton, Marantec, Nice Flo, Roger |
| Dynamic (KeeLoq) | Rolling counter encrypted with KeeLoq algorithm | BFT Mitto, DoorHan, Aprimatic, Hormann EcoStar |
| Dynamic (other) | Rolling counter with manufacturer-specific encoding | CAME Atomo, Nice FloR-S, Somfy Telis, FAAC SLH |
| Pseudo-Dynamic | Appears rolling but derived from a fixed base | AN-Motors AT4, CAME TWEE |
