Unleashed Firmware is a fork of the official [Flipper Zero firmware](https://github.com/flipperdevices/flipperzero-firmware). It is the most stable custom build available, incorporating new features and improvements while remaining fully compatible with the official API and app ecosystem.

  This software is intended solely for experimental and educational purposes. Use only within the bounds of applicable law. This project is developed independently and is not affiliated with Flipper Devices.

  DarkFlippers/unleashed-firmware is the only official page of this project. There are no paid, premium, or closed-source versions. Anyone claiming otherwise is a scammer.
</Warning>

## Why Unleashed instead of official firmware?

The official firmware enforces regional Sub-GHz transmission restrictions and ships without several quality-of-life features. Unleashed removes those restrictions, adds support for dozens of new protocols, ships extra bundled apps, and backports community improvements — all without breaking compatibility with the official app catalog or mobile apps.

Key differences from official firmware:

- **No Sub-GHz region locks** — all CC1101-supported frequencies work out of the box
- **Rolling code protocol support** — capture and replay supported rolling code remotes (FAAC SLH, BFT Mitto, Nice Flor-S, Somfy Telis, CAME Atomo, and more)
- **Extra bundled apps** — Sub-GHz Bruteforce, Sub-GHz Remote, LFRFID Fuzzer, iButton Fuzzer, and more included in every release
- **Quality-of-life improvements** — customizable Flipper name, clock on desktop, battery percentage, BadKB (BadUSB over Bluetooth), and additional keyboard layouts
- **NFC enhancements** — extra Mifare Classic keys, EMV public data parser, transit card parsers for Troika, Kazan, Metromoney, and others
- **Fully open source** — every release is free; there will never be a paywall

## Releases

Unleashed Firmware has two release channels:

- **Stable releases** — published on [GitHub Releases](https://github.com/DarkFlippers/unleashed-firmware/releases) and the [Telegram channel](https://t.me/unleashed_fw)
- **Dev builds** — built automatically from the `dev` branch; available at [dev.unleashedflip.com](https://dev.unleashedflip.com) and the [Telegram dev channel](https://t.me/kotnehleb)

<Note>
  Dev builds are not release-ready and may contain bugs. Use stable releases for everyday use.
</Note>
